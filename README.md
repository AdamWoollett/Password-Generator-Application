This is a simple web application which can be used by a user to generate random strings that a user can use as a password when signing up for a service.
The user can change the desired password length and whether the password will include numbers and symbols.
The web application has a strength meter to evaluate the strength of the password output, along with a dialogue box recommending the user selects at least one of numbers of symbols to include if none are selected.
There is also functionality to copy the generated password to the user's clipboard at the push of a button, with the program highlighting the test to show this has occurred.
User experience was a key consideration when building this project.
This project was made using the Vite framework with Tailwind CSS.
